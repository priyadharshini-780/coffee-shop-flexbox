# Coffee Shop Website ☕

## Overview
This is a responsive Coffee Shop website created using HTML5 and CSS3. The project demonstrates the use of CSS Flexbox to build a clean, modern, and responsive layout.

## Features
- Hero Section
- Coffee Menu Cards
- Customer Reviews
- Contact Section
- Responsive Design
- Hover Effects
- Flexbox Layout

## Technologies Used
- HTML5
- CSS3

## CSS Flexbox Concepts
- display: flex
- flex-direction
- justify-content
- align-items
- flex-wrap
- gap
